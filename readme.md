# Piper Voice API

Голосовой API для ИИ-ассистента: TTS (Piper) + STT (Vosk).

## Локальный запуск
```bash
docker build -t piper-api .
docker run -p 8000:8000 piper-api
# Открой: http://localhost:8000/docs

Когда всё заработает, можно:
Добавить логирование запросов (чтобы видеть, что спрашивают гости)
Настроить кеширование частых ответов (экономия токенов)
Подключить UptimeRobot, чтобы избежать холодного старта
Добавить второй голос (irina) и переключать по параметру
